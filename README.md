# Gjimnazi "Abdulla Keta" Website

Website zyrtar i gjimnazit "Abdulla Keta" në Tiranë, Shqipëri.

## Teknologjitë e përdorura

- React 
- TypeScript
- Express 
- PostgreSQL
- Framer Motion
- Tailwind CSS

## Funksionalitetet

- Sistem i plotë administrimi për stafin
- Navigim i thjeshtë për nxënësit
- Seksione të ndryshme për klube, aktivitete dhe lajme
- Bibliotekë digjitale me mundësi kërkimi
- Pamje responsive për të gjitha pajisjet
- Formularë kontakti me ruajtje në databazë

## Instalimi dhe ekzekutimi

Për të ekzekutuar projektin lokalisht:

```bash
# Instalimi i varësive
npm install

# Fillimi i serverit të zhvillimit
npm run dev
```

## Struktura e projektit

```
/client - Kodi i frontend-it në React + TypeScript
/server - API-të e backend-it në Express
/shared - Skemat dhe definicionet e përbashkëta
/public - File-t statike dhe asetet
```
